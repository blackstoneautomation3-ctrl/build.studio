# Build.Studio v2

> Next.js 15 + Tailwind CSS + TypeScript

## Stack
- **Next.js 15.2.1** with App Router + Turbopack
- **React 19**
- **TypeScript**
- **Tailwind CSS 3.4**
- **Lucide React** icons
- **Playfair Display** + **Sora** + **JetBrains Mono** fonts

## Quick Start
```bash
npm install
npm run dev
```
Open [http://localhost:3000](http://localhost:3000)

## Deploy
```bash
git init && git add . && git commit -m "launch"
git remote add origin YOUR_GITHUB_URL
git push -u origin main
```
Import on [vercel.com](https://vercel.com) → Deploy ✅

## Design System
| Token | Value |
|---|---|
| Background | `#080808` (obsidian) |
| Surface | `#111111` (ink) |
| Accent | `#e8b86d` (amber) |
| Text | `#f2f2f0` (snow) |
| Muted | `#666666` (ash) |
